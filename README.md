# Expense Tracker

A simple Python CLI Expense Tracker that stores data in a local JSON file. Track your daily expenses, view summaries, and manage your spending — all from the command line.

## Features

- Add Expense
- View Expenses
- Calculate Total Expenses
- Delete Expense
- Edit Expense
- Search Expense
- Category Summary
- Exit

## Tech Stack

- Python 3
- JSON (for data storage)

## How to Run

```bash
python Expense_Tracker.py
```
